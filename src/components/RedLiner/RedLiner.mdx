# RedLiner

Used to show style attritbutes on any display component.

## Props

<Props of={RedLiner} />

## Usage

### Default Options

The default options.

<Playground>
    <RedLiner>
      <button style={buttonStyle}>
        Measure Me
      </button>
    </RedLiner>
</Playground>

### Config Prop

Customize color of lines, sections to display, and properties to show in the InfoBox.

<Playground>
  <RedLiner config={{ color: 'blue', infoOpts: ['borderRadius', 'color'] }}>
    <button style={buttonStyle}>Measure Me</button>
  </RedLiner>
</Playground>

<br />

Remove the InfoBox component with `displayOpts`.

<Playground>
  <RedLiner config={{ displayOpts: ['height', 'width'] }}>
    <input style={inputStyle}></input>
  </RedLiner>
</Playground>

### Show on Hover

Use the prop `showOnHover` to hide RedLiner until the child component is hovered.
<Playground>
  <RedLiner showOnHover>
    <button style={buttonStyle}>helllllo</button>
  </RedLiner>
</Playground>

### Maximum Customization

<Playground>
  <RedLiner config={{ color: 'purple', displayOpts: ['height', 'info'], infoOpts: ['borderRadius', 'color'] }} showOnHover>
    <button style={buttonStyle}>Measure Me</button>
  </RedLiner>
</Playground>

### Components Prop

It's possible to replace and modify certain components in RedLiner. In this example, we replace the default InfoBox component with a custom version.

<Playground>
  <RedLiner components={{ InfoBox: CustomInfoBox }}>
    <button style={buttonStyle}>Measure Me</button>
  </RedLiner>
</Playground>

##### Custom InfoBox Component Example

```jsx
import React from 'react';
import { Popper } from 'react-popper';
import styles from 'styles/index.scss';

export const CustomInfoBox = (props: any) => (
  <Popper placement="right">
    {({ ref, style, placement, arrowProps }) => (
      <div className={styles.info} ref={ref} data-placement={placement} style={style}>
        <div className={styles.arrow} ref={arrowProps.ref} style={arrowProps.style} />
          <h6 style={{ fontSize: '16px' }}>Custom Info</h6>
        <span>p: {props.computedStyle.padding}</span>
      </div>
    )}
  </Popper>
);
```

#### Alternate Custom InfoBox
```jsx
import React from 'react';
import Select, { defaultComponents } from 'redliner';

const CustomInfoBox = (props: any) => {
  return (
    <div style={{ border: '1px dotted blue' }}>
      <defaultComponents.InfoBox {...props} />
    </div>
  );
}
```
